📊 Customer Churn Prediction System

A Machine Learning-based web application that predicts whether a customer will churn or stay based on their usage behavior and service interaction data.

This project demonstrates an end-to-end ML pipeline including data preprocessing, model training, and deployment via a Streamlit dashboard.

## 🧠 Problem Statement

Customer churn is a major issue in subscription-based businesses.  
The goal of this project is to predict whether a customer will leave the service or continue using it based on behavioral and usage data.

---

## 📊 Features

- Predict customer churn (Yes / No)
- Displays churn probability score
- Interactive Streamlit dashboard
- Business insights based on user input
- Real-time prediction system

---

## 🛠️ Tech Stack

- Python 🐍
- Pandas 📊
- Scikit-learn 🤖
- Streamlit 🌐
- Joblib 📦

---

## 🤖 Machine Learning Model

- Algorithm: Logistic Regression / Random Forest
- Data preprocessing: StandardScaler
- Problem type: Binary Classification
- Output: Churn (1) / No Churn (0)
- Model accuracy: ~85%
